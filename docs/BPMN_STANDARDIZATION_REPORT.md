## Technical Notes

### FEEL to Java Expression Conversion
Zeebe uses FEEL expressions prefixed with `=`:
- `=patientId` → `${patientId}`
- `=encounter.status` → `${encounter.status}`

### External Task Workers
All external tasks now use `camunda:topic` attribute which matches the Python worker subscriptions defined in ADR-003.

### DMN Integration
Business rule tasks with `camunda:decisionRef` will invoke DMN tables via the CIB Seven embedded DMN engine.

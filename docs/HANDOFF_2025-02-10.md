## 2. Workflow Protocol (CRITICAL - Read ADR-013)

**File:** `docs/ADRs/013-claude-flow-swarm-intelligence.md`

### Agent Role: PLANNER, NOT CODER

The human uses Claude-flow swarm for implementation. Your job:
1. **Pre-task routine** - Execute hooks, store scope in memory
2. **Prepare swarm command** - Display for human to copy-paste
3. **Wait for completion** - Human runs swarm in separate terminal
4. **Post-task routine** - Record outcome, train patterns, commit

### Pre-Task Commands
```bash
# 1. Pre-task hook
npx @claude-flow/cli@latest hooks pre-task --description "Wave X Description" --task-id "wave-x-id" --namespace healthcare-platform

# 2. Store scope in memory
npx @claude-flow/cli@latest memory store --key "wave-x-scope" --value "scope details..." --namespace healthcare-platform
```

### Swarm Command Template
```bash
npx @claude-flow/cli@latest swarm spawn \
  --leader architect \
  --agents coder,reviewer,tester \
  --task "Task description" \
  --consensus 3 \
  --namespace healthcare-platform
```

### Post-Task Commands
```bash
# 1. Record outcome
npx @claude-flow/cli@latest hooks post-task --success --task "Wave X" --message "outcome summary"

# 2. Train (may error - ignore)
npx @claude-flow/cli@latest hooks intelligence train --source "path/*.py" --namespace healthcare-platform

# 3. Store completion
npx @claude-flow/cli@latest memory store --key "wave-x-complete" --value "summary..." --namespace healthcare-platform

# 4. Git commit and push
git add . && git commit -m "feat(scope): Wave X - Description" && git push
```
